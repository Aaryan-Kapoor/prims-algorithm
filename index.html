Pseudocode Trace

1. Initialize MST as empty
2. Select starting node
3. Add starting node to MST
4. WHILE MST has fewer than N nodes:
   5. Find minimum edge (u,v,w)
   6. WHERE u is in MST
   7. AND v is NOT in MST
   8. Add v to MST
   9. Add edge (u,v) to MST
   10. Update total weight
11. END WHILE
12. MST is complete!
